# 중소기업 취업자 소득세 감면 페이지 기획

## 1. 페이지 개요

중소기업에 취업한 청년, 고령자, 장애인, 경력보유여성을 위한 소득세 감면 제도 안내 페이지

### 페이지 구조 (2개 페이지)

| 페이지           | URL                             | 구성                      |
| ---------------- | ------------------------------- | ------------------------- |
| 제도 소개 페이지 | `/sme-tax-reduction`            | 제도 소개 + 양식 다운로드 |
| 감면 기간 계산기 | `/sme-tax-reduction/calculator` | 감면 기간 계산기          |

### 페이지 간 링크

- 제도 소개 페이지 → 계산기 페이지: "감면 기간 계산하기" CTA 버튼
- 계산기 페이지 → 제도 소개 페이지: "제도 자세히 보기" 링크

## 2. 제도 소개 섹션

### 2.1 메인 타이틀
**중소기업 취업자 소득세 감면**
> 중소기업에 취업한 청년, 고령자, 장애인, 경력보유여성이라면 소득세를 감면받을 수 있어요.

### 2.2 제도 인지도 안내
- 제도 시행 10년 이상 경과
- 회사에서도 잘 몰라 안내해 주지 못하는 경우 많음
- 직접 신청해야만 받을 수 있는 혜택
- 많은 분들이 놓치고 있음

### 2.3 감면 대상자 (근로자 요건)

#### 대상자 유형별 조건

| 유형 | 연령 조건 | 감면율 | 감면기간 |
|------|----------|--------|----------|
| 청년 | 15세~34세 | 90% | 5년 |
| 고령자 | 60세 이상 | 70% | 3년 |
| 장애인 | 연령 무관 | 70% | 3년 |
| 경력보유여성 | 연령 무관 | 70% | 3년 |

※ 청년의 경우 병역 복무기간(최대 6년)은 연령 계산에서 차감

#### 제외 대상
- 해당 회사의 임원 또는 최대주주
- 최대주주의 배우자나 자녀 등 특수관계인
- 일용근로자
- 건강보험료(직장가입자) 납부이력이 없는 분

### 2.4 감면 대상 기업 (중소기업 요건)

1. **｢중소기업기본법｣ 제2조에 따른 중소기업**
   - 업종별로 중소기업으로 인정받는 규모 기준이 다름
   - [중소기업 확인 바로가기](https://sminfo.mss.go.kr) 링크 제공

2. **｢조세특례제한법 시행령｣ 제27조에 규정된 감면대상 업종**
   - 중소기업 요건을 충족하더라도 특정 업종은 혜택 제한
   - [법령 확인 바로가기](https://www.law.go.kr) 링크 제공

### 2.5 혜택 상세

#### 감면 한도
- **연간 최대 200만 원**까지 감면
- 평균 순세액감면액: 약 48만 원

#### 이직/재취업 시
- 다른 중소기업으로 이직하거나 재취업해도 혜택 유지
- 감면 기간은 **최초 소득세 감면 신청 회사의 취업일**로부터 계산

**예시**: 청년 근로자가 첫 직장에서 2년 4개월 감면 → 1년 휴직 → 이직 시 남은 1년 8개월 동안 감면 가능

### 2.6 신청 방법

#### 근로자 (재직 중)
1. **중소기업 취업자 소득세 감면 신청서** 작성
2. 회사 담당자에게 제출 (파일 또는 인쇄물)
3. ⏰ 제출 기한: 취업일 기준 **다음 달 말일까지**

#### 회사 담당자
1. **중소기업 취업자 소득세 감면 대상 명세서** 작성
2. 관할 세무서 또는 홈택스에 제출
3. ⏰ 제출 기한: 감면 신청 받은 달의 **다음 달 10일까지**

#### 근로자 (퇴직 후)
- 신청서를 주소지 관할 세무서에 직접 제출 가능

### 2.7 경정청구 안내
- 혜택을 받을 수 있었는데 신청하지 않아 더 낸 세금이 있다면
- **경정청구**를 통해 돌려받을 수 있음
- 경정청구: 잘못 낸 세금을 바로잡아 다시 신고하는 제도

## 3. 감면 기간 계산기

### 3.1 입력 항목

| 항목 | 입력 형식 | 필수 여부 | 비고 |
|------|----------|----------|------|
| 생년월일 | YYYY.MM.DD | 필수 | DatePicker 사용 |
| 취업일 (입사일) | YYYY.MM.DD | 필수 | DatePicker 사용 |
| 취업자 유형 | Select | 필수 | 청년/고령자/장애인/경력보유여성 |
| 군입대일 (소집일) | YYYY.MM.DD | 조건부 | 청년 + 병역이행자만 |
| 군전역일 (소집해제일) | YYYY.MM.DD | 조건부 | 청년 + 병역이행자만 |

### 3.2 계산 로직

#### (5) 중소기업에 취업한 날 연령
```
취업일 연도 - 생년월일 연도
단, 취업일이 생일 이전이면 -1
```

**예시**:
- 생년월일: 1991.04.05
- 취업일: 2025.09.15
- 계산: 2025 - 1991 = 34년, 09.15 > 04.05 이므로 생일 지남
- 결과: **34년 05월 11일** (정확한 일수 계산)

#### (6) 병역근무기간 (청년만, 6년 한도)
```
군전역일 - 군입대일
최대 6년(72개월)까지만 인정
```

**예시**:
- 군입대일: 2013.04.01
- 군전역일: 2014.12.31
- 결과: **1년 08월 27일** (또는 1년 9개월)

#### (7) 병역근무기간 차감 후 연령 (청년만)
```
(5) - (6)
```

**예시**:
- 취업 시 연령: 34년 05월 11일
- 병역기간: 1년 08월 27일
- 결과: **32년 08월 15일** (만 32세)

※ 병역 차감 후 연령이 34세 이하면 청년 감면 대상

#### (8) 감면 시작일
```
취업일 (2012.1.1 이후 소득세 감면을 받은 최초 취업일)
```

#### (9) 감면 종료일

**일반적인 경우**:
```
청년: 시작일 + 5년 - 1일 (해당 월의 말일)
그 외: 시작일 + 3년 - 1일 (해당 월의 말일)
```

**병역이행 후 1년 이내 동일 중소기업 복직 시**:
```
복직일 + 2년 - 1일 (해당 월의 말일)
단, 복직일이 최초 취업일부터 5년 이내여야 함
5년 초과 시: 최초 취업일 + 7년 - 1일 (해당 월의 말일)
```

**예시**:
- 감면 시작일: 2025.09.15
- 취업자 유형: 청년
- 감면 종료일: 2030.09.30 (5년 후 해당 월 말일)

### 3.4 자격 판정 로직

```typescript
function checkEligibility(input: {
  birthDate: Date;
  employmentDate: Date;
  workerType: 'youth' | 'senior' | 'disabled' | 'careerWoman';
  militaryStartDate?: Date;
  militaryEndDate?: Date;
}): EligibilityResult {

  // 1. 취업 시 연령 계산
  const ageAtEmployment = calculateAge(input.birthDate, input.employmentDate);

  // 2. 청년인 경우 병역기간 차감
  let adjustedAge = ageAtEmployment;
  let militaryPeriod = null;

  if (input.workerType === 'youth' && input.militaryStartDate && input.militaryEndDate) {
    militaryPeriod = calculatePeriod(input.militaryStartDate, input.militaryEndDate);
    // 최대 6년까지만 차감
    const maxMilitaryYears = 6;
    militaryPeriod = Math.min(militaryPeriod, maxMilitaryYears);
    adjustedAge = ageAtEmployment - militaryPeriod;
  }

  // 3. 자격 판정
  let isEligible = false;
  let reductionRate = 0;
  let reductionPeriod = 0;

  switch (input.workerType) {
    case 'youth':
      isEligible = adjustedAge >= 15 && adjustedAge <= 34;
      reductionRate = 90;
      reductionPeriod = 5;
      break;
    case 'senior':
      isEligible = ageAtEmployment >= 60;
      reductionRate = 70;
      reductionPeriod = 3;
      break;
    case 'disabled':
    case 'careerWoman':
      isEligible = true;
      reductionRate = 70;
      reductionPeriod = 3;
      break;
  }

  // 4. 감면 기간 계산
  const startDate = input.employmentDate;
  const endDate = calculateEndDate(startDate, reductionPeriod);

  return {
    isEligible,
    ageAtEmployment,
    militaryPeriod,
    adjustedAge,
    reductionRate,
    reductionPeriod,
    startDate,
    endDate,
    annualLimit: 2000000 // 200만원
  };
}
```
