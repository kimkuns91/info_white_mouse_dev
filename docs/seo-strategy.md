# SEO 전략 문서

> **목표**: 단순히 검색 결과 상단 노출을 넘어, '구매력 있는 근로자'가 스스로 검색하고 들어오게 만드는 것

---

## 1. 키워드 전략: 의도(Intent) 기반 키워드 점유

### 1.1 정보형 키워드 (Top of Funnel - 유입 유도)

연봉 계산기는 경쟁이 치열하므로, 보다 구체적이고 실무적인 키워드를 서브 페이지로 생성

| 키워드 예시 | 전략 |
|-------------|------|
| 2026 연봉 실수령액 표 | 전문가 가이드 텍스트 배치 |
| 포괄임금제 연차수당 계산 | 구글 스니펫 노출 유도 |
| 퇴직금 중간정산 조건 | 500자 이상의 상세 가이드 |
| 알바 주휴수당 자동계산 | 실용적 유틸리티 제공 |

### 1.2 욕망/결핍 키워드 (Middle of Funnel - 락인)

재무 분석과 연계된 키워드로 디블럭 잠재 고객 유입

| 키워드 예시 | 연계 콘텐츠 |
|-------------|-------------|
| 사회초년생 첫차 유지비 | 연봉별 적정 차량 유지비 가이드 |
| 연봉 4000 자동차 추천 | 디블럭 서사로 자연스럽게 연결 |
| 출퇴근 왕복 2시간 건강 관리 | 차량 내 공기질 관리 콘텐츠 |
| 차 안 냄새 제거 방법 | 디블럭 제품 연계 |

## 2. 기술적 SEO 최적화

### 2.1 구조화 데이터 (Structured Data) 마크업

```json
{
  "@context": "https://schema.org",
  "@type": "WebApplication",
  "name": "2026 연봉 계산기",
  "applicationCategory": "FinanceApplication",
  "operatingSystem": "Web"
}
```

**적용 스키마**:
- `SoftwareApplication` 또는 `WebApplication`: 계산기 기능
- `FAQPage`: 자주 묻는 질문 (예: "연봉 계산 시 비과세 항목은 무엇인가요?")

### 2.2 Core Web Vitals 관리

| 지표 | 목표 | 방법 |
|------|------|------|
| LCP (최대 콘텐츠 칠) | < 2.5초 | 계산기 로직 실행 속도 최적화 |
| FID (첫 입력 지연) | < 100ms | JavaScript 번들 최적화 |
| CLS (레이아웃 시프트) | < 0.1 | 이미지/폰트 사전 로드 |

### 2.3 모바일 퍼스트 인덱싱

- 출퇴근길 지하철에서 모바일로 확인하는 유저 타겟
- 모바일 UI의 가독성과 버튼 클릭 범위 최적화

## 3. 콘텐츠 서사 SEO (On-Page SEO)

### 3.1 Semantic HTML 구조

**Bad**:
```html
<h1>2026 연봉 계산기</h1>
```

**Good**:
```html
<h1>2026 연봉 계산기: 당신의 실수령액과 자산 가치까지 분석하는 AI 스토리텔러</h1>
```

### 3.2 이미지 Alt 텍스트 전략

**Bad**:
```html
<img src="dblock-banner.jpg" alt="디블럭 배너">
```

**Good**:
```html
<img src="dblock-banner.jpg" alt="연봉 협상 성공 후 나를 위한 선물, 프리미엄 자동차 에어컨 필터 디블럭">
```

## 4. 세그먼트별 랜딩 페이지 최적화

### 4.1 URL 구조 설계

| URL | 용도 |
|-----|------|
| `/calculator/salary` | 기본 연봉 계산기 |
| `/analysis/commuter-health` | 장거리 출퇴근러 전용 |
| `/analysis/first-car-manager` | 첫 차 보유자 전용 |
| `/tips/commuter-health` | 출퇴근 미세먼지 관리법 |
| `/tips/car-maintenance-cost` | 에어컨 필터 셀프교체 비용 |

### 4.2 세그먼트 전용 메타 태그

각 세그먼트 페이지마다 검색 엔진에 노출되는 설명문(Meta Description)을 다르게 설정

**예시**:
> "왕복 2시간 출퇴근, 당신의 연봉만큼 폐 건강도 중요합니다. 지금 무료 건강-재무 리포트를 확인하세요."

## 5. 외부 신호 강화 (Off-Page SEO)

### 5.1 커뮤니티 바이럴 소스

타겟 커뮤니티:
- 블라인드 (직장인)
- 펨코 (재테크)
- 뽐뿌 (쇼핑/정보)

**전략**: 자연스러운 백링크(Backlink) 유도
> "이번에 연봉 계산기 UI 진짜 깔끔한 거 찾음"

### 5.2 SNS 공유 최적화 (OG Tag)

유저가 재무 분석 결과를 카톡/인스타에 공유할 때, 디블럭의 가치가 포함된 매력적인 썸네일이 나오도록 설정

## 6. 구글 vs 네이버 최적화

### 6.1 구글 (Google) - 권위와 전문성(E-A-T)

#### 서치콘솔 설정
- Sitemap.xml 등록
- URL 검사 도구로 새 페이지 색인 생성 요청
- 구조화 데이터 JSON-LD 적용

#### 기술적 요소
- HTTPS 보안 (SSL) 필수
- PageSpeed Insights 90점 이상 유지

### 6.2 네이버 (Naver) - 신뢰도와 실시간성

#### 서치어드바이저 설정
- '리포트 > 사이트 최적화' 모든 항목 초록색 체크
- 수집 주기 '빠르게' 설정 (연초 연봉 협상 시즌 대비)

#### 콘텐츠 전략
**Bad**:
```
whitemouse | 연봉 계산기
```

**Good**:
```
2026 연봉 계산기 - 실수령액 확인하고 디블럭으로 차 안 건강까지 챙기세요
```

## 7. 메타 태그 세트

### 7.1 메인 연봉 계산기 페이지

```html
<title>2026 연봉 실수령액 계산기 - 부장님 몰래 보는 팩트 체크 | info.whitemouse.dev</title>
<meta name="description" content="내 월급만 제자리인가요? 2026년 최신 비과세 및 세율을 반영한 가장 정확한 연봉 계산기. 위트 있는 재무 분석 리포트와 함께 당신의 진짜 가치를 확인하세요.">
```

### 7.2 자동차 보유 근로자 전용 페이지

```html
<title>연봉별 적정 자동차 유지비 가이드: 내 차는 자산일까, 짐일까? | info.whitemouse.dev</title>
<meta name="description" content="연봉 5,000만 원, 아반떼 유지비는 얼마가 적당할까요? 데이터로 본 자동차 관리의 정석. 기름값보다 중요한 '차 안 공기질' 관리 비결과 디블럭 혜택을 확인하세요.">
```

### 7.3 사회초년생 전용 페이지

```html
<title>사회초년생 첫 월급 관리의 기술 - 지름신을 이기는 위트 있는 가이드</title>
<meta name="description" content="첫 월급 받으셨나요? 축하합니다! 지옥철 출근길을 견딘 당신을 위해, 자산 형성과 가심비 소비를 동시에 잡는 스토리텔링 재무 설계를 시작해보세요.">
```

## 8. SEO 시나리오 예시

1. **검색**: 유저가 네이버에 "2026 연봉 실수령액" 검색
2. **노출**: `info.whitemouse.dev` 결과가 "부장님 몰래 보는 2026 연봉표"라는 위트 있는 제목으로 노출
3. **유입**: 유저가 들어와 연봉 계산 완료
4. **분석 리포트**: "당신의 연봉 수준은 '상위 15%'입니다. 하지만 차 안 공기질은...?" 멘트와 함께 디블럭 추천 배너 노출
5. **신뢰 쌓기**: 하단에 '전문적인 자동차 필터 관리 가이드' 텍스트 배치
