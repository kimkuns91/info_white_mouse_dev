# 연봉 계산 로직 문서

2026년 기준 연봉/월급 실수령액 계산 로직을 설명합니다.

## 입력값

| 항목 | 설명 | 타입 |
|------|------|------|
| `amount` | 연봉 또는 월급 금액 | number |
| `isMonthly` | 월급 기준 여부 (false면 연봉) | boolean |
| `nonTaxable` | 비과세 수당 (식대 등) | number |
| `dependents` | 부양가족 수 (본인 포함, 최소 1) | number |
| `children8to20` | 8세~20세 자녀 수 (자녀세액공제용) | number |

## 계산 흐름

### 1단계: 월 총급여 산출

```
월 총급여 = 연봉인 경우 ? 연봉 / 12 : 월급
과세표준 = max(0, 월 총급여 - 비과세 수당)
```

### 2단계: 국민연금

```
연금기준소득 = min(max(과세표준, 400,000), 6,370,000)
국민연금 = floor(연금기준소득 × 0.0475 / 10) × 10
```

- 기준소득월액 상한(637만원)과 하한(40만원) 적용
- 10원 단위 절사

### 3단계: 건강보험

```
건강보험 = floor(과세표준 × 0.03595 / 10) × 10
```

- 과세표준에 3.595% 적용
- 10원 단위 절사

### 4단계: 장기요양보험

```
장기요양보험 = floor(건강보험 × 0.1314 / 10) × 10
```

- 건강보험료의 13.14%
- 10원 단위 절사

### 5단계: 고용보험

```
고용보험 = floor(과세표준 × 0.009 / 10) × 10
```

- 과세표준에 0.9% 적용
- 10원 단위 절사

### 6단계: 근로소득세 (간이세액표 조회)

**⚠️ 핵심 포인트**
- 누진세율로 직접 계산하지 않습니다!
- **국세청 간이세액표**를 룩업하여 사용합니다.
- 간이세액표에는 근로소득공제, 기본공제, 특별소득공제 등이 이미 반영되어 있습니다.

#### 간이세액표 구조

```typescript
interface TaxBracket {
  min: number;  // 월급여 하한 (천원 단위)
  max: number;  // 월급여 상한 (천원 단위)
  taxes: number[];  // 부양가족 1~11명별 세액 (원)
}
```

#### 조회 방법

```typescript
function getIncomeTax(monthlySalary: number, dependents: number): number {
  // monthlySalary: 비과세 제외한 월급여 (원)
  // dependents: 공제대상가족 수 (본인 포함, 최소 1)

  const salaryInThousand = Math.floor(monthlySalary / 1000);

  // 간이세액표에서 구간 찾기
  const bracket = TAX_TABLE.find(row =>
    salaryInThousand >= row.min && salaryInThousand < row.max
  );

  if (!bracket) return 0;

  // 부양가족 수에 해당하는 세액 반환 (최대 11명)
  const familyIndex = Math.min(dependents, 11) - 1;
  return bracket.taxes[familyIndex];
}
```

#### 월급여 1천만원 초과 시 계산 공식

간이세액표는 월 1천만원까지만 제공되며, 그 이상은 공식으로 계산:

| 월급여 구간 | 계산 공식 |
|-------------|-----------|
| 1,000만원 초과 ~ 1,400만원 이하 | (1천만원 세액) + (초과금액 × 98% × 35%) + 25,000원 |
| 1,400만원 초과 ~ 2,800만원 이하 | (1천만원 세액) + 1,397,000원 + (1,400만원 초과금액 × 98% × 38%) |
| 2,800만원 초과 ~ 3,000만원 이하 | (1천만원 세액) + 6,610,600원 + (2,800만원 초과금액 × 98% × 40%) |
| 3,000만원 초과 ~ 4,500만원 이하 | (1천만원 세액) + 7,394,600원 + (3,000만원 초과금액 × 40%) |
| 4,500만원 초과 ~ 8,700만원 이하 | (1천만원 세액) + 13,394,600원 + (4,500만원 초과금액 × 42%) |
| 8,700만원 초과 | (1천만원 세액) + 31,034,600원 + (8,700만원 초과금액 × 45%) |

### 7단계: 자녀세액공제

8세~20세 자녀가 있는 경우 소득세에서 공제:

| 자녀 수 | 월 공제액 |
|---------|-----------|
| 1명 | 12,500원 |
| 2명 | 29,160원 |
| 3명 이상 | 29,160원 + (자녀수 - 2) × 25,000원 |

```
소득세 = max(0, 간이세액표 세액 - 자녀세액공제)
```

### 8단계: 지방소득세

```
지방소득세 = floor(소득세 × 0.1 / 10) × 10
```

- 소득세의 10%
- 10원 단위 절사

### 9단계: 최종 계산

```
총 공제액 = 국민연금 + 건강보험 + 장기요양보험 + 고용보험 + 소득세 + 지방소득세
실수령액 = 월 총급여 - 총 공제액
```

## 계산 예시

### 연봉 5,400만원, 비과세 40만원, 부양가족 1인

**입력:**
- 연봉: 54,000,000원
- 비과세: 400,000원
- 부양가족: 1명

**계산:**
1. 월 총급여: 54,000,000 / 12 = 4,500,000원
2. 과세표준: 4,500,000 - 400,000 = 4,100,000원

**4대보험:**
- 국민연금: 4,100,000 × 4.75% = 194,750원
- 건강보험: 4,100,000 × 3.595% = 147,390원
- 장기요양: 147,390 × 13.14% = 19,370원
- 고용보험: 4,100,000 × 0.9% = 36,900원

**세금:**
- 과세표준: 4,100천원 (간이세액표 4,100~4,120 구간)
- 소득세: 209,310원 (간이세액표 조회, 부양가족 1명)
- 지방소득세: 209,310 × 10% = 20,930원

**결과:**
- 총 공제액: 628,650원
- 실수령액: 4,500,000 - 628,650 = **3,871,350원**

## 참고사항

1. 본 계산기는 **국세청 간이세액표(2024.2 개정)**를 사용합니다.
2. 실제 세액은 연말정산 시 확정됩니다.
3. 비과세 항목(식대, 자가운전보조금 등)은 4대보험 및 세금 계산에서 제외됩니다.
4. 간이세액표는 근로소득공제, 기본공제, 특별소득공제가 이미 반영되어 있습니다.
5. 2026년용 간이세액표는 2026년 2월 발표 예정이며, 현재는 2024년 버전을 사용합니다.
