# 연봉 계산 로직 문서

2026년 기준 연봉/월급 실수령액 계산 로직을 설명합니다.

## 입력값

| 항목 | 설명 | 타입 |
|------|------|------|
| `amount` | 연봉 또는 월급 금액 | number |
| `isMonthly` | 월급 기준 여부 (false면 연봉) | boolean |
| `nonTaxable` | 비과세 수당 (식대 등) | number |
| `dependents` | 부양가족 수 | number |
| `childrenUnder20` | 20세 이하 자녀 수 | number |

## 2026년 4대보험 요율

| 항목 | 요율 | 비고 |
|------|------|------|
| 국민연금 | 4.75% | 근로자 부담분 (총 9.5%, 2026년 인상) |
| 건강보험 | 3.595% | 근로자 부담분 (총 7.19%, 2026년 인상) |
| 장기요양보험 | 13.14% | 건강보험료의 13.14% (2026년 인상) |
| 고용보험 | 0.9% | 근로자 부담분 (동결) |

## 기준소득월액 상/하한 (2025.7~2026.6)

| 항목 | 금액 |
|------|------|
| 국민연금 상한 | 6,370,000원 |
| 국민연금 하한 | 400,000원 |
| 건강보험 상한 | 110,332,300원 |

## 계산 흐름

### 1단계: 월 총급여 산출

```
월 총급여 = 연봉인 경우 ? 연봉 / 12 : 월급
과세표준 = max(0, 월 총급여 - 비과세 수당)
```

### 2단계: 국민연금

```
연금기준소득 = min(max(과세표준, 400,000), 6,370,000)
국민연금 = floor(연금기준소득 × 0.0475 / 10) × 10
```

- 기준소득월액 상한(637만원)과 하한(40만원) 적용
- 10원 단위 절사

### 3단계: 건강보험

```
건강보험 = floor(과세표준 × 0.03595 / 10) × 10
```

- 과세표준에 3.595% 적용
- 10원 단위 절사

### 4단계: 장기요양보험

```
장기요양보험 = floor(건강보험 × 0.1314 / 10) × 10
```

- 건강보험료의 13.14%
- 10원 단위 절사

### 5단계: 고용보험

```
고용보험 = floor(과세표준 × 0.009 / 10) × 10
```

- 과세표준에 0.9% 적용
- 10원 단위 절사

### 6단계: 근로소득세

연간 과세소득에 따른 누진세율 적용:

| 과세표준 구간 | 세율 | 누진공제액 |
|---------------|------|------------|
| 1,400만원 이하 | 6% | 0원 |
| 1,400만원 ~ 5,000만원 | 15% | 840,000원 |
| 5,000만원 ~ 8,800만원 | 24% | 6,240,000원 |
| 8,800만원 초과 | 35% | 15,360,000원 |

```
연간과세소득 = 과세표준 × 12

if (연간과세소득 <= 14,000,000)
  연간소득세 = 연간과세소득 × 0.06
else if (연간과세소득 <= 50,000,000)
  연간소득세 = 840,000 + (연간과세소득 - 14,000,000) × 0.15
else if (연간과세소득 <= 88,000,000)
  연간소득세 = 6,240,000 + (연간과세소득 - 50,000,000) × 0.24
else
  연간소득세 = 15,360,000 + (연간과세소득 - 88,000,000) × 0.35

부양가족공제 = (부양가족 수 + 20세 이하 자녀 수) × 150,000
월소득세 = max(0, (연간소득세 - 부양가족공제) / 12)
소득세 = floor(월소득세 / 10) × 10
```

### 7단계: 지방소득세

```
지방소득세 = floor(소득세 × 0.1 / 10) × 10
```

- 소득세의 10%
- 10원 단위 절사

### 8단계: 최종 계산

```
총 공제액 = 국민연금 + 건강보험 + 장기요양보험 + 고용보험 + 소득세 + 지방소득세
실수령액 = 월 총급여 - 총 공제액
```

## 계산 예시

### 연봉 4,000만원, 비과세 20만원, 부양가족 1인 (본인)

**입력:**
- 연봉: 40,000,000원
- 비과세: 200,000원
- 부양가족: 1명

**계산:**
1. 월 총급여: 40,000,000 / 12 = 3,333,333원
2. 과세표준: 3,333,333 - 200,000 = 3,133,333원

**4대보험:**
- 국민연금: 3,133,333 × 4.75% = 148,830원
- 건강보험: 3,133,333 × 3.595% = 112,640원
- 장기요양: 112,640 × 13.14% = 14,800원
- 고용보험: 3,133,333 × 0.9% = 28,190원

**세금:**
- 연간과세소득: 3,133,333 × 12 = 37,599,996원
- 연간소득세: 840,000 + (37,599,996 - 14,000,000) × 15% = 4,379,999원
- 부양가족공제: 1 × 150,000 = 150,000원
- 월소득세: (4,379,999 - 150,000) / 12 = 352,500원
- 소득세: 352,500원
- 지방소득세: 352,500 × 10% = 35,250원

**결과:**
- 총 공제액: 692,210원
- 실수령액: 3,333,333 - 692,210 = 2,641,123원

## 참고사항

1. 본 계산기는 간이세액표 기반의 추정치를 제공합니다.
2. 실제 세액은 연말정산 시 확정됩니다.
3. 비과세 항목(식대, 자가운전보조금 등)은 4대보험 및 세금 계산에서 제외됩니다.
4. 부양가족 공제는 간이 계산을 위한 추정 공제액입니다.
